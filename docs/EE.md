# EASTER EGGS

## Twitch

-   easter eggs can be found in these docs by those curious enough to be looking here
-   if a user tries to `!give` XP to themselves, the bot will instead take those points from the user and give them to Hist
-   if 3 different users manage to combo a specific sequence of emotes, they get a reward (e.g. user 1: emote A + user 2: emote B + user 3: emote C = win)
-   if a user manages to post chat messages at a precise rate (e.g. 1 msg every 5 mins), they earn extra XP for the duration of their streak (e.g. instead of each message granting 10 xp, they get 15 xp)
